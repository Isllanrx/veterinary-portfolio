# Portfólio Profissional — Vitor Casagrande

Este é o portfólio profissional de **Vitor Casagrande**, estudante sênior de Medicina Veterinária, focado em saúde pública, inspeção sanitária e bem-estar animal. O projeto foi desenvolvido com tecnologias web de ponta para garantir performance, SEO e acessibilidade internacional.

## Demonstração & Idiomas

O portfólio está disponível em três idiomas, acessíveis através do seletor no topo da página:
- **Português (Nativo)**
- **English (Intermediate)**
- **Español (Intermedio)**

## Tecnologias Utilizadas

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router & Turbopack)
- **Estilização:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Componentes:** [Shadcn UI](https://ui.shadcn.com/) (Radix UI)
- **Internacionalização:** Sistema customizado via React Context (i18n)
- **Ícones:** [Lucide React](https://lucide.dev/)
- **Monitoramento:** [Vercel Analytics](https://vercel.com/analytics)
- **Deploy:** [Vercel](https://vercel.com/)

## Principais Funcionalidades

- **Design Minimalista & Profissional:** Interface "Clear" focada em legibilidade e autoridade.
- **Timeline de Carreira:** Exibição dinâmica da trajetória profissional com destaque para métricas (horas acumuladas).
- **SEO 100%:** Metadados dinâmicos, OpenGraph para redes sociais e manifest PWA.
- **Responsividade Total:** Adaptado para Desktop, iOS e Android.
- **PWA Ready:** Instalável como aplicativo em dispositivos móveis.
- **Formulário de Contato:** Integração direta com WhatsApp para conversão rápida.

## Configuração Local

Para rodar este projeto em sua máquina:

1. **Instale as dependências:**
   ```bash
   npm install
   # ou
   pnpm install
   ```

2. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

3. **Build de produção:**
   ```bash
   npm run build
   ```
